# Color Sensor

## Wiring

The color sensor is wired to the I2C port on the rio using 4 conductor wire. See I2C for a pinout.

## Specs

|                        |         |
| ---------------------- | ------- |
| Connection Type        | I2C     |
| I2C Address            | 0x52    |
| Operating Voltage      | 3.3VDC  |
| Proximity Sensor Range | 1-10 cm |
