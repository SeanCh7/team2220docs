# Limit Switch

## Wiring/Pinout

The limit switch is wired to the DIO ports on the RoboRIO, most commonly using 4 conductor wire. The usage of 4 conductor wire allows for the connection of up to 2 limit switches on one wire run.

|      |              |
| ---- | ------------ |
| N.O. | DIO/Signal   |
| COM  | DIO/Positive |
