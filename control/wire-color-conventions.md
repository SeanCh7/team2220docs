# Wire Color Conventions

The following is a list of common wire colors and their meaning/usage.

| Color  | Usage                |
| ------ | -------------------- |
| Red    | Positive/VCC/Anode   |
| Black  | Negative/GND/Cathode |
| Yellow | Can High (CANH)      |
| Green  | Can L (CANL)         |
| White  | Signal               |
| Green  | Signal               |
